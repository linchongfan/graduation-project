## 使用说明

### 运行程序

```bash
# 进入源代码目录
cd src

# 运行混合蚁群算法
python hybrid_aco.py

# 运行基本蚁群算法
python main.py
```

### 生成流程图

```bash
# 进入工具目录
cd tools

# 生成所有流程图
java -jar plantuml-1.2026.4.jar ../docs/flowcharts/*.pu

# 生成单个流程图
java -jar plantuml-1.2026.4.jar ../docs/flowcharts/05_hybrid_algorithm.pu
```
