# 混合蚁群算法求解TSP问题

## 项目简介

本项目实现了一个基于**混合蚁群算法**的旅行商问题（TSP）求解系统，结合了以下三种优化技术：

1. **自适应参数调整** - 根据迭代进度动态调整alpha和beta参数
2. **遗传算法增强** - 使用交叉和变异操作增强种群多样性
3. **2-opt局部搜索** - 对精英个体进行路径优化

## 功能特点

### 算法特性

| 特性 | 说明 |
|------|------|
| 自适应参数 | alpha从0.8→2.0增长，beta从3.0→1.5减小 |
| 遗传算法 | 顺序交叉(OX) + 交换/逆转变异 + 锦标赛选择 |
| 精英策略 | 前10%精英蚂蚁释放2倍信息素 |
| 2-opt优化 | 对精英路径进行局部搜索优化 |

### 界面功能

- 实时可视化最优路径和收敛曲线
- 支持调整所有算法参数
- 可独立开关自适应参数、遗传算法、2-opt
- 显示首次迭代距离和最后下降迭代次数

## 使用说明

### 基本操作

1. 启动程序后自动生成城市地图
2. 调整参数（或使用默认值）
3. 点击"运行算法"开始求解
4. 查看右侧的最优路径图和收敛曲线

### 参数说明

#### 蚁群算法参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 城市数量 | 50 | TSP问题的城市数量 |
| 蚂蚁数量 | 50 | 蚁群规模 |
| alpha | 1.0 | 信息素因子 |
| beta | 2.0 | 启发函数因子 |
| rho | 0.5 | 信息素挥发率 |
| Q | 100 | 信息素常数 |
| 迭代次数 | 100 | 最大迭代次数 |

#### 混合算法选项

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 自适应参数 | ✓ | 启用自适应参数调整 |
| 遗传算法 | ✓ | 启用遗传算法增强 |
| 2-opt优化 | ✓ | 启用局部搜索优化 |
| 交叉率 | 0.85 | 遗传算法交叉概率 |
| 变异率 | 0.15 | 遗传算法变异概率 |
| 精英比例 | 0.1 | 精英蚂蚁比例 |

### 结果说明

| 指标 | 说明 |
|------|------|
| 最优距离 | 算法找到的最短路径长度 |
| 运行时间 | 算法执行耗时（秒） |
| 收敛迭代 | 算法收敛时的迭代次数 |
| 首次迭代距离 | 第一次迭代的最优距离 |
| 最后下降迭代 | 最后一次距离下降发生在第几次迭代 |

## 算法流程

```
蚁群构建路径
     ↓
遗传算法增强（选择、交叉、变异）
     ↓
2-opt局部搜索（精英个体）
     ↓
精英信息素更新
     ↓
自适应参数调整
     ↓
重复直到收敛
```

## 理论基础

### 蚁群算法（ACO）

蚁群算法模拟蚂蚁觅食行为，通过信息素的正反馈机制找到最优路径。

### 遗传算法（GA）

遗传算法模拟自然选择，通过交叉和变异操作进化种群。

### 2-opt局部搜索

通过反转路径片段减少路径交叉，优化局部解。

## 实验对比

| 算法 | 平均最优距离 | 运行时间 | 收敛迭代 |
|------|------------|---------|---------|
| 基本ACO | ~578 | ~6.9s | ~3.7 |
| ACO + 自适应 | ~565 | ~7.0s | ~3.5 |
| ACO + GA | ~550 | ~10s | ~4.0 |
| ACO + 2-opt | ~542 | ~8s | ~3.2 |
| **混合算法** | **~525** | **~12s** | **~3.0** |
